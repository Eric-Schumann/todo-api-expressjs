Todo API Documentation

There are three urls you will be working with in this API. 

  • http://localhost/api/todo
  • http://localhost/api/todos
  • http://localhost/api/todos/:todo_id

The HTTP methods aviailable are below.

  • GET
    • All Todos
      http://localhost/api/todos
    • Specific Todo
      http://localhost/api/todos/:todo_id
  • POST
    http://localhost/api/todo
  • PUT
    http://localhost/api/todos/:todo_id
  • DELETE
    http://localhost/api/todos/:todo_id

PUT & POST

  • PUT - Updates the specific todo to is_completed = true.
  • POST - Createds a new todo with the following attributes
    • title - "Name of the todo."
    • body - "Details of the todo"

DELETE

DELETE - This will do a hard delete from the database. Once removed, the todo will no longer be available or recoverable.
